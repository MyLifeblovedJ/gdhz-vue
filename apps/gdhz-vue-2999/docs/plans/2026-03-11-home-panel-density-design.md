# Home Panel Density Design

> 日期：2026-03-11
> 范围：`apps/gdhz-vue-2999`

## 目标

继续微调首页右列与海堤风险的信息密度：

- `海岸观测` 在小屏幕下更紧凑，减少视频卡之间的上下留白
- `海岸观测` 的卡片高度与网格行高保持一致，避免网格被拉高后卡片本体仍固定高度
- `海堤风险` 的城市信息继续放在断点后面，但视觉样式恢复为原先次级信息的弱化表达

## 设计结论

- 将 `海岸观测` 的卡片高度与网格间距纳入首页视口分档
- `CoastalObservationPanel` 改为通过 CSS 变量接收卡片高度与间距，并让视频卡填满网格行
- `SeawallRiskPanel` 保持城市信息在标题行内，但拆成单独的次级文本节点，使用接近原位置行的颜色和字号

## 验收标准

- 小屏幕下 `海岸观测` 的卡片更紧凑，卡片之间上下空白明显减少
- 大屏幕下右侧仍能铺满，但不会出现夸张的卡片内留白
- `海堤风险` 的城市文字看起来是辅助信息，不抢海堤名和断点名
