# 预警组件V1最终规格（首版）

## 1. 范围
- 页面：`apps/welcome-5173`
- 数据源：`/api/screen/bootstrap`（Mock/API 同构字段）
- 目标：实现市级预警摘要与地图高亮联动

## 2. 交互与展示规则
1. 新增预警摘要组件，位置在头部下方左侧，展示：
- 市级主预警（同市合并后）
- 发布时间相对值（如“15分钟前”）
- 截止时间（`YYYY-MM-DD HH:mm`）

2. 地图支持多个城市高亮并存：
- 每个城市独立边界实体
- 同时存在多个 active 预警时全部展示

3. 同市主灾种判定规则：
- 优先风暴潮（按等级取最高）
- 否则按灾种优先级：海啸 > 海浪 > 赤潮 > 其他

4. 缩放显示规则（`warningSpec.zoomBoundaryKm`）：
- 相机高度 `> 150km`：显示填充 + 边界 + 中心灾种图标（或回退圆点）
- 相机高度 `<= 150km`：仅显示边界

5. 撤销规则：
- `warningFeed.items[].status = cancelled` 时，对应城市摘要与高亮立即消失

6. 图标规则：
- 从项目可用目录读取（示例：`/icons/disaster/*.png`）
- 图标缺失自动回退中心圆点
